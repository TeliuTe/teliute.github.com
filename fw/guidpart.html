EFI系统分区设定盘符 取消盘符（来源：http://jingyan.baidu.com/article/454316ab6efc05f7a7c03a33.html）

给efi分区添加分区号，以管理员身份运行cmd，输入：

　　diskpart

　　sel disk 0 (此处为你的efi分区所在磁盘号，单硬盘为数字0)

　　list part （列出磁盘中的所有分区，以便查出efi的分区号）

　　sel part 1 (此处为EFI分区号，输入上一行查出的id号，第一个是分区1，类型一般为“主要”)

　　set id=ebd0a0a2-b9e5-4433-87c0-68b6b72699c7 （分区类型id设为基本数据类型，如上表所示）

　　assign letter=y (y为分配的盘符，任意)

取消efi设定的分区号

以管理员身份运行cmd，输入：

　　diskpart

　　sel disk M(M为你的efi分区所在磁盘号，一般为数字0)

　　list part

　　sel part x (x为EFI分区分区号，根据上面列出输入，一般是分区1，类型为“主要”)

　　set id=c12a7328-f81f-11d2-ba4b-00a0c93ec93b （EFI系统分区ID）
