第1课 学会设置Fedora系统

1、登录用户：d2018d，前进，跳过，开始使用，关闭窗口


2、先用拼音，点左上角，活动，主文件夹，在空白处点右键，新建文件夹 7-3



3、点左上角“文件-首选项”


4、在出来第面板里，点第二个“行为”，选择“单击”，和“每次”，关闭窗口；


5、右上角，设置，用户，图标，用户名，密码抄本子上（如果灰色下次再改）


6、火狐主页：火狐，右边三道杠，首选项，主页设为校园网


7、设置输入法：左上角，活动，fc，fcitx配置，只留下汉语和搜狗拼音，关闭窗口；


8、左下角点开竖条按钮，右键，设置，去掉”隐藏状态栏“的勾，
    点“确定”，拖到最上边黑条里面，时间的左边；


9、鼠标手势：校园网，点软件下载，linux，


10、再点 firegesture， Allow - install - restart now ，只点一下
    别点多，等3秒后，浏览器重启；


11、点左上角“活动”，搜索tw，点“优化工具”，设置下面的选项；


12、在出来第窗口左边，选择“打字”，
    右边找到“杀死X”，右边选三个组合键，这是重启动的；


13、在左边找到“扩展”，右边最下面Window list，点成“开启”，
    窗口下边就会出来任务栏；


14、再在左边点“桌面”，右边“开启”，打勾“主文件夹”，


15、再在左边点“窗口”，右边“开启”最大化、最小化，关闭窗口；



16、再到校园网，点 软件下载，linux，indicator-shutdown..，  
      瞄准点右键，链接另存为，桌面，保存，关闭


17、最小化浏览器，回到桌面，瞄准 ind..，点右键 - 属性
  在第二个“权限”里，打勾“允许作为程序运行”

　本节学习了的基础知识，如果你成功地完成了练习，请继续学习下一课内容；
