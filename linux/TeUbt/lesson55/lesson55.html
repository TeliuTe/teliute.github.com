五十五、dconf Edit 配置编辑器　返回目录　下一课

　使用它，可以编辑一些系统设置选项，类似的还有 gconf-edit 程序；

　1、dconf Edit 配置编辑器

　1）点击主按钮，在搜索中输入  dconf ，打开程序，或者依次点“主按钮、所有程序、过滤结果、辅助功能、dconf Edit”；

　

　

　2）要想在通知区域显示 Fcitx 输入法图标，在左侧依次点击“desktop－unity－panel”；

　

　

　3）在右边出来的属性值(Value)里单击，在后面添加上 ,'fcitx' 然后按回车确定，注销重新登录即可；

　

　

　4）其他的程序图标也类似，如果想恢复原来的值，点击右下角的“恢复默认 Set to Default”按钮，

　如果想对所有程序开放显示，将值改成 ['all']

　

　本节学习了dconf Edit 配置编辑器的基础知识，如果你成功地完成了练习，请继续学习下一课内容；
