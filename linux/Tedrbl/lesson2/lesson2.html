　接下来安装无盘相关的服务；

　1、安装无盘服务

　1）打开终端，输入命令 /opt/drbl/sbin/drblsrv -i 在出来密码提示中，输入自己的密码然后按回车键，默认的选项是大写字母，输入 n 或者直接按回车键，不通过网络安装 Linux；

　

　

　2）接下来是否安装序列控制台，输入 n 或直接按回车键；

　

　

　3）接下来询问客户机使用什么样的内核，输入2和服务器相同级别的CPU；

　

　

　4）然后开始下载软件包，稍等提示是否更新升级操作系统，输入 n 或者按回车键；

　

　

　5）在询问客户机安装内核时，输入1跟服务器相同的内核；

　

　

　6）耐心等待安装完成，如果提示有错误就仔细检查一下，一般 Ubuntu Lts 版本和 Debian 支持的比较好；

　

　

　本节学习了安装无盘服务的基础知识，如果你成功地完成了练习，请继续学习下一课内容；
