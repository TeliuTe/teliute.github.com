为网页添加图片

译者：TeliuTe 来源：基础教程网

在网页中插入一幅图片

您可以在网页中插入 GIF, JPEG, BMP 和 PNG (便携网络图片)格式图片， 您还可以使用它们作为 图片链接。当您插入一幅图片， KompoZer 将在页面中保存一个图片的位置。

注意: 如果您想把页面发布到网上，最好不要用 BMP 格式图片。

小提示: 在插入图片前，最好先保存或发布一下网页，这将允许 KompoZer 自动使用图片的相对地址。

插入图片:

  1. 点击您想插入图片的位置。
  2. 点击工具栏上的图片按钮 ，或者点插入菜单选择图片，出来插入图片属性对话框。
  3. 输入图片的位置和文件名，或者点浏览按钮选择一幅本机或网络上的图片。
  4. 输入简短的描述作为图片的替代文本，这会出现在纯文本浏览器中(和其他浏览器一样)， 在图片正在加载或禁用图片时，会显示这个文本。

    另外，您也可以选择不包括替代文本。

  5. 如果需要， 在 图片属性 对话框里， 点击其他标签，进行相应设置(例如对齐方式)。

小提示: 快速插入一幅图片: 拖动一幅图片到页面中。

小提示: 在所有图片段落后面插入一个换行，在插入菜单中选图片下方换行。

编辑图片属性

一旦您插入一幅图片到页面中，您就可以编辑它的属性设置层，像高度、宽度、空白， 以及文字对齐等。如果您还没有打开图片属性对话框，请按下面步骤操作:

编辑选中图片的属性:

  1. 双击图片，或者选种它再点工具栏上的图片按钮 ， 显示图片属性对话框。
  2. 点击位置标签编辑这些属性:
    • 图片位置: 输入图片文件名和位置，点浏览按钮选择本机或网络上的图片。
    • URL 是相对地址: 选中此项， KompoZer 转换网址为一个相对地址， 这在您计划发布页面到网络上，让其他人访问时非常有用。使用相对地址您可以把所有的文件归在一起，保持相对路径， 不用考虑它们在本机和网络上的位置差异。

      不选中此复选框，KompoZer 会转换所有地址为全称(绝对)地址。 一般在引用网络上的图片时，使用绝对地址(不存储在您的本机中)。

      如果您还没有保存或发布页面，您必须先保存一下，以便激活此复选框 (如果您在消息窗口打开图片属性对话框，这个复选框也是不可用的)。

    • 替代文本: 输入一个位于原来图片位置的文本，例如，一个标题或者简要的图片描述。 这是一个好习惯，对于使用纯文本浏览器，或禁用图片浏览器的访问者，很有帮助。
    • 不使用替代文本: 如果图片不需要替代文本，或者您不想包括此项，则选中它。
  3. 点击大小标签编辑下列属性:
    • 实际大小: 选中此项，恢复到图片原始大小。
    • 自定义大小: 选中此项指定新的高度和宽度，用百分比或像素。这个设置不影响图片的原始文件，仅仅是在页面显示。
    • 保持纵横比: 如果您改变了图片的大小，选择此项将保持图片的外观比例(以便图像不失真)，选中此项您只需更改高度或宽度中的一项即可。
  4. 点外观标签可以编辑下列的选项:
    • 边距: 指定图片四周的空白，当它周围有文字相邻时，您也可以指定图片四周有一个实线边框，并指定边框线的大小(像素)，0 是无边框。
    • 对齐文本到图片: 如果图片周围有许多文本，您可以选择一种文本和图片的对齐方式。
    • 图片映射: 点击删除按钮，删除所有图片映射设置。
  5. 点链接标签设置下列属性:
    • 输入一个网页位置: 如果您想用这幅图片定义一个超链接，输入本地或远程地址，或者从下拉列表中选择一个锚点、标题。点浏览按钮从本机或网络上选择一个文件。
    • URL 是相对地址:选中此项， KompoZer 转换网址为一个相对地址， 这在您计划发布页面到网络上，让其他人访问时非常有用。使用相对地址您可以把所有的文件归在一起，保持相对路径，不用考虑它们在本机和网络上的位置差异。

      不选中此复选框，KompoZer 会转换所有地址为全称(绝对)地址。 一般在引用网络上的图片时，使用绝对地址(不存储在您的本机中)。

      如果您还没有保存或发布页面，您必须先保存一下，以便激活此复选框 (如果您在消息窗口打开图片属性对话框，这个复选框也是不可用的)。

    • 在链接图片周围显示边框: 选中此项，在图片周围显示高亮颜色。
  6. 要应用额外的属性或 JavaScript 事件，点击高级编辑按钮，显示 高级属性编辑器.
  7. 点确定完成更改。
