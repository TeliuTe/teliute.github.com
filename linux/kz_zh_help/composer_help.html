用 KompoZer 创建网页

译者：TeliuTe 来源：基础教程网

KompoZer 帮助您创建自己的网页并发布到网络上，使用 KompoZer 您不需要懂得太多，使用它就像使用词语一样简单。

您可以使用工具栏上的按钮，从而很方便地添加列表、表格、图片、超链接到其他页面，设置颜色、字体样式等。 您可以在创建时看到页面发布时的外观，您也可以很方便地与其他人分享您的文档，不必担心他们使用何种浏览器， 或者他们使用的是哪一种电子邮件处理程序。

创建一个新页面

创建一个新页面

KompoZer 是一个 HTML(超文本标记语言)编辑器，它允许您 创建和编辑网页。KompoZer 是一个 WYSIWYG(所见即所得) 编辑器，因此您在创建的时候，就可以看到页面发布的样子。它不需要您懂得 HTML，因为绝大部分 HTML 功能都可以用工具栏上的按钮和菜单完成。

KompoZer 也可以让你编辑 HTML 源代码，如果你愿意的话。 要查看或编辑 HTML 源代码，打开查看菜单，然后选择 HTML 源代码，或者 在下边的编辑模式工具栏中，点击 <HTML> 源代码标签。

要创建一个网页，可以使用下面讲述的任一种方法，当您打开一个页面，您 可以在一个文本编辑器里添加或编辑您想要的文本。

要在 KompoZer 里创建新页:

  • 点击 KompoZer 工具栏上的新建按钮。

保存 HTML 文件到本地磁盘上:

  1. 打开窗口菜单，选择 KompoZer，会出来 KompoZer 窗口
  2. 点击 文件 菜单然后选择 打开 命令，出来打开文件对话框。
  3. 在磁盘上找到您想编辑的文件。
  4. 点 打开 按钮，在 KompoZer 窗口中就显示了这个文件。

要编辑网页:

  1. 打开 KompoZer
  2. 点 文件 菜单，然后选 打开位置 命令，出来一个打开网络位置的对话框。
  3. 输入您想编辑页面的网址(例如， http://www.mozilla.org) 然后点创建按钮。您会看到页面显示在 KompoZer 中。

KompoZer 也给出一个创建新页面的选项，使用 XHTML 格式 来创建一个 XHTML 文档，在 创建新文档或模板 对话框里勾选 "创建 XHTML 文档"。

小提示: 在 KompoZer 窗口里，您可以通过打开文件菜单，选择 最近访问页面，在打开的最近访问文件列表中，选择您想要打开的文件。

标签 - 编辑多个文档

KompoZer 允许您同时编辑多个文档，而不用在一个个窗口之间切换， KompoZer 提供了 标签 的特性。 在一个 KompoZer 窗口里可以同时打开多个文档，分别放在不同的标签中。

新标签可以通过按组合键 Ctrl+T 打开，或者在文件菜单里， 点 新建标签。 对于打开的多个标签，每一个都是相对独立的，每个 文档有一个标题显示，例如，如果该页面需要保存，就在对应的标签用一个用红色图标指示。

保存并浏览页面

您可以用 HTML 或纯文本格式保存 KompoZer 文档。保存 HTML 格式将 保留文档的格式，例如文本样式(像粗体或斜体)、表格、链接和图片等。保存成纯文本 将删除所有 HTML 标记，只保留文本内容。

保存成 HTML 文件:

  • 打开文件菜单，然后选择保存命令，或者点击工具栏上的保存按钮。

    如果您还没有给页面设定标题，KompoZer 会提示您完成。 KompoZer 会在窗口的标题栏上显示页面标题，如果您添加了书签，文档的页面标题 也会出现在您的书签列表中。

    然后 KompoZer 提示您输入文件名并指定文件的保存位置。确保您保留了 .html 扩展名，如果编辑保存的是 XHTML 文档，文件的扩展名应当是 .html。 文件的扩展名是 .html 但使用了XHTML DOCTYPE 则不会认为是 xhtml 文档。

要更改文件名或保存文件到另外一个位置:

  • 点菜单 文件 > 另存为 然后选择不同的文件名或位置。

当您用 KompoZer 保存页面，所有的页面部分 (HTML, 图片和其他文件，像声音和样式表)，都会保存到您的硬盘上。如果您仅仅想保存页面的 HTML 部分， 您需要更改 KompoZer 保存页面的首选项，更多相关信息，请参阅 KompoZer 首选项。

如果一个图片位置是绝对地址(以"http://"开头)而您又正连接到互联网， 那么您还可以在 KompoZer 中看到这幅图片并访问它。但是，如果图片位置是 相对地址(以"file:///"开头)，那么您就不能在本地文件中看到图片。

保存纯文本文件:

  1. 点 文件 菜单，然后选择 导出文本文件。
  2. 输入文件名然后指定文件的保存位置。

注意: 图片不会保存到纯文本格式文件中。

小提示: 您可以在文件菜单中选择恢复命令， 复原到最近一次保存的文件，但您之后的更改将会丢失。

要在浏览器中查看页面以便测试您的链接:

  • 点文件菜单，然后选择浏览页面(或者点击工具栏上的浏览按钮)。如果文件还没有保存， KompoZer 会提示输入页面标题、文件名和位置。浏览器窗口会挡在 KompoZer 窗口的前面。
