使用样式表设置页面样式

译者：TeliuTe 来源：基础教程网

使用层叠样式表

层叠样式表(CSS)有三种方法设计 HTML 文档:

  1. 使用内联样式。
  2. 使用内部样式表。
  3. 使用外部样式表。

KompoZer 有一个内置的样式表编辑器叫做 CaScadeS。 CaScadeS 既可以设计内部样式表，也可以设计外部样式表， 相对于内联样式，内部或外部样式表可以让页面内容与样式信息分离。

要在文档中使用样式，启动CaScadeS，可以点击工具菜单，然后选择 CSS 编辑器， CaScadeS 的左面板有一排按钮，第一个按钮有下拉列表，允许选择样式类型:

注意: 如果没有建立样式表，系统会自动新建一个内部样式表。

用 KompoZer 创建样式表

使用 CaScadeS 可以创建内部样式表，也可以创建外部样式表，要创建内部样式表:

  1. 点击左上角样式表下拉按钮，选择内部样式表
    • (可选)输入样式表的设备列表和标题信息。
  2. 点击创建样式表按钮。

要创建一个外部样式表:

  1. 点击左上角样式表下拉按钮，选择外部样式表
  2. 在右边的面板里输入样式表的地址(URL)，如果它不存在，将会在本地系统里创建这个样式表文件。
    • (可选)输入样式表的设备列表和标题信息。
  3. 如果这是一个替代样式表，打勾选中“选中创建一个替代样式表”
  4. 点击创建样式表按钮。

小提示: 在关联本地样式表前记住先保存一下 html 文档，同样关闭 CSS 编辑器后也要保存文档。

小提示: 如果样式表没有立即下载，请点击面板左侧的刷新按钮。

创建样式表规则

在创建一个或多个样式表后，就可以分别创建规则，要为某一个样式表创建或修改规则， 在左侧的面板中点击选中这个样式表，右边的常规面板中将会显示样式表的细节， 要创建一个新规则:

  1. 在面板左上角直接点击规则按钮。
  2. 右面板会显示几个选项，选择创建哪一种规则，各个选项分别是:
    • 应用到所有同一类元素的样式(在下面选择或输入名称)
    • 应用到所有元素的样式(在下面输入名称)
    • 应用到特定 ID 属性元素的样式
    • 自定义样式规则
  3. 输入规则的名称。
  4. 点击创建样式规则按钮。

在右边的面板中，可以用样式标签来定义规则(文本、背景、边框、盒子和声音)。 要查看一个规则的所有定义，在左侧面板选中它，然后点右边的“常规”标签， 常规标签显示所有当前应用到规则的定义。
