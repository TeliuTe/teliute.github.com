在 KompoZer 里创建超链接

译者：TeliuTe 来源：基础教程网

用 KompoZer 创建页内超链接

要创建一个内面内部的超链接，这样读者就可以快速在不同章节之间跳转，您必须要创建 锚点 (目标位置)，然后创建指向这个锚点的，锚点也叫书签.

  1. 点击您想要放置锚点的行开头，把光标插入点放在那儿，或者选中一些文本。
  2. 点击工具栏上的锚点按钮 ，或者点插入菜单，选择锚点，出来锚点属性对话框。
  3. 在名称文本框输入锚点的名称，注意不能重名(30字符以内)，如果使用了空格，会被转换为下划线(_)，如果在第一步选中了一些文字，那么这儿已经包含了一个默认的名称。
  4. 点确定按钮，文档中对应位置出现一个锚点标记:

创建能够跳转到锚点的链接:

  1. 选中要创建链接的文本或图片，这将指向锚点。
  2. 点击工具栏上的链接按钮，或者点插入菜单，然后选择链接，出来一个链接属性对话框。
    • 如果您要链接到其他页面，点浏览选择这个文件。
    • 如果您要创建指向锚点的链接，点下拉按钮，列表中会显示可用的锚点。
    • 如果您要创建指向标题的链接(如标题1 - 标题6)，从下拉列表中选择当前可用的标题。
  3. 点确定完成。

注意: 要测试您创建的链接，点文件菜单选择浏览页面，然后点击测试链接。

小提示: 如果您不是首次创建锚点，您可以使用链接对话框创建链接到已有的标题。

创建指向其他页面的链接

您在页面内可以创建链接到本机的页面，或者是网络工作组的页面，也可以是远程的网页。

小提示: 在创建链接前，最好先保存或发布一下页面 这样 KompoZer 可以自动创建地址为相对地址。

创建指向其他页面的链接:

  1. 点击您想要创建链接的位置，把光标插入点放在那儿，或者选中一些文本或图片。
  2. 点击工具栏上的链接按钮，出来一个链接属性对话框。
  3. 设置链接:
    • 链接文本: 如果您选中了图片文件或者一些文本，选中的文本或文件 选中的文本或文件会出现在这里，否则您就要在这里输入一段文本，作为链接显示的文本。
    • 链接位置: 输入本地文件名或远程页面的地址，如果您不太清楚文件路径，可以点浏览按钮，在本机中查找文件。对于远程页面网址，您可以采用复制粘贴的方法。另外您可以从下拉列表中选择当前页面的锚点或标题。
    • URL 是相对地址: 选中此项，KompoZer 转换网址为一个相对地址， 这在您计划发布页面到网络上，让其他人访问时非常有用。使用相对地址您可以把所有的文件归在一起，保持相对路径，不用考虑它们在本机和网络上的位置差异。

      不选中此复选框，KompoZer 会转换所有地址为全称(绝对)地址。 一般在引用网络上的图片时，使用绝对地址(不存储在您的本机中)。

      如果您还没有保存或发布页面，您必须先保存一下，以便激活此复选框。

  4. 要应用额外的属性或 JavaScript 事件，点击高级编辑按钮，显示 高级属性编辑器。
  5. 点确定完成。
  6. 要测试您创建的链接，点浏览按钮，然后测试链接是否正确。

小提示: 您可以从其他窗口拖动链接到页面，以便快速添加一个链接 例如，您可以从网页、书签和邮件窗口中，点击并拖动链接到页面。您也可以用右键点击链接( 在 Mac OS 中是 Control+点击)，然后在出来的菜单中选择复制链接地址，然后粘贴到链接属性 对话框的位置文本框里。

在链接中添加 XFN 信息

在创建指向其他网页的链接时，KompoZer 提供了一个特性，可以包括 XFN (XHTML 友情链接)信息。XFN 使用超级链接来表达 人们之间的相互关系，要添加 XFN 信息:

  1. 在您想添加 XFN 信息的链接上点右键。
  2. 选择“链接属性”。
  3. 在“链接属性”对话框里，点击“更多属性”。
  4. 在“友情链接”段，选择合适的项目，从而添加 XFN 信息。
  5. 点“确定”完成。
  6. 点“保存”按钮，保存文档的更改。

注意: 一些 XFN 属性是单选按钮，只能选一项，另外一些是复选框，可以选多项。

使用图片作链接

您给页面中的图片添加链接，比如 JPEG, GIF, PNG 图片文件。当点击图片时， 浏览器窗口打开链接指向的页面。

  1. 在页面中选择一幅图片。
  2. 点击工具栏上的链接按钮 ，或者点插入 菜单选择链接。
  3. 在链接属性对话框里，设置链接图片到 页内的锚点或标题，或者链接到 本地或远程页面。

小提示: 在浏览器中拖动一幅图片到 KompoZer 窗口，将会 复制图片地址和它指向的链接。

注意: 要删除链接图片周围的蓝色边框，打开链接属性对话框， 点击链接标签，不选中“显示链接图片周围的边框”。

删除或取消链接

删除链接:

  1. 选中链接文本(通常是蓝色带下划线)或图片。
  2. 打开格式菜单然后选择删除链接命令。

要取消一个链接，使得链接文本恢复到普通文本:

  1. 在链接上点一下，把光标插入点放置其中。
  2. 打开格式菜单，选择取消链接命令。
