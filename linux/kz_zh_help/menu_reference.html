KompoZer 菜单索引

译者：TeliuTe 来源：基础教程网

这里讲述了 KompoZer 窗口的各个菜单项。

文件

新建

创建一个新的 KompoZer 标签或窗口。

打开文件...

显示打开对话框，您可以选择打开一个磁盘上的文件。 在 KompoZer 中，您可以查看 HTML 文件、HTML 模板文件和文本文件。

打开网络位置...

显示打开网络位置对话框，您可以输入一个网址或选择打开一个本地文件。

最近打开

显示最近编辑文件列表。

关闭

关闭当前窗口。

全部关闭

关闭当前窗口打开的所有标签。

保存

保存当前标签页打开的文件。

另存为...

用一个不同的文件名保存当前编辑的页面。

保存并设置字符编码

在显示“另存为”对话框前，给出一个更改“字符编码”的对话框选项。

发布

打开当前编辑文档的发布对话框。

发布为...

打开当前编辑文档的发布对话框，操作跟发布相同。

恢复

撤销最近一次保存以来的所有更改，恢复到最近一次保存时的文档。

导出为文本文档...

导出当前文档的内容为文本文档。

在浏览器中预览

在默认的浏览器中查看当前页面

页面设置...

打开页面设置选项对话框，配置打印选项。

打印预览

打开一个窗口，显示文档打印到纸张上的效果。

打印...

显示打印对话框，您可以在这里选择打印机，打印份数等等，点 确定 开始打印。

退出

关闭所有 KompoZer 窗口，包括打开的各个标签，完全退出 KompoZer。 如果您还要未保存的文档，KompoZer 将询问您是否真的要退出，防止偶尔关闭窗口。

编辑

撤销

撤销当前文档的最近一次操作，如果您又不想撤销了，可以使用重做命令。

重做

重做最近一次撤销的操作。

剪切

剪切文档中选中的文本，存储到剪贴板中，在别处使用粘贴命令粘贴文本。

复制

复制选中的文本到剪贴板中，在别处使用粘贴命令粘贴文本。

粘贴

把剪贴板中使用剪切或复制的文本，粘贴到当前光标位置。

粘贴(不含格式)

只粘贴剪贴板中的文本到当前位置，格式信息被丢弃。

删除

删除当前选中的文本。

全选

选中当前文档的所有文本。

查找和替换...

显示一个对话框，您可以在里面熟人要查找和替换的字符串，只在当前文档中进行搜索。

查找下一个

执行 查找和替换... 命令，继续查找下一个匹配项，只在当前文档中进行搜索。

查找上一个

搜索上一个 查找和替换... 命令的匹配项。

检查拼写

打开拼写检查器，对当前页文本进行拼写检查。

从模板中分离

把当前文档从模板中分离出来，保存为一个 html 页面。

网站发布设置

打开站点管理器配置或更改发布站点设置。

[ 返回到开头 ]

查看

显示/隐藏

常用工具栏

显示或隐藏常用工具栏，包含常用的按钮(锚点、链接、图片等)。

格式工具栏

显示或隐藏格式工具栏，包含各种格式按钮。

编辑模式工具栏

显示或隐藏编辑模式工具栏，它在 KompoZer 窗口的底部，状态栏的上面。您可以用它切换不同视图方式，比如 HTML Tags view, HTML 源代码 等视图。

状态栏

显示或隐藏 KompoZer 窗口底部的状态栏。

标尺

显示或隐藏水平标尺和垂直标尺。

自定义工具栏

主工具栏

打开定制主工具栏面板，您可以从面板拖动按钮到主工具栏，以增加新的按钮，反之删除一个按钮。

格式工具栏

打开定制格式工具栏面板，您可以从面板拖动按钮到主工具栏，以增加新的按钮，反之删除一个按钮。

站点管理器

显示或隐藏 KompoZer 窗口里的 站点管理器。

普通编辑模式

切换到普通编辑模式下。

HTML 标记

切换到 HTML 标记视图模式。

HTML 源代码

切换到 HTML 源代码视图模式。

预览

切换到预览视图模式。

回车标记

显示或隐藏回车标记。

块标记

显示或隐藏块级别元素的边框。

文本缩放

缩小

使得页面中的文本减小，您也可以使用鼠标来操作(请参阅文本大小一章)。要恢复文本原来的大小，按组合键 Ctrl+0。

放大

增大显示页面中的文本，您也可以使用鼠标来操作，按住 Ctrl 键，然后转动鼠标滚轮，或者按 + 或 -，增大减小文本大小。要恢复文本原来的大小，按组合键 Ctrl+0。

字符编码

允许您手动更改网页里的字符编码。 KompoZer 通常自动检测完成。

插入

插入菜单允许您向当前文档中插入各种 HTML 元素。

图片...

打开插入图片对话框，插入一幅图片到当前文档。

表格...

打开插入表格对话框，插入一个表格到当前文档。

表单...

链接...

打开链接对话框，创建一个超级链接，如果之前选中里一段文本，选中的文本将变成链接，您也可以通过点击“更多属性”，定义友情链接(XFN)。

锚点...

打开锚点对话框，在光标处插入一个锚点。

水平线

在当前位置插入一条水平线。

HTML...

打开一个对话框，您可以输入 HTML 源代码，这也可以插入注释、脚本代码等等。

字符和符号...

打开一个对话框，选择指定的字符或符号，插入到当前位置。

目录表...

打开插入目录表窗口，在当前位置插入目录表，目录表(TOC)可以用标题1到标题6,一个 <div> 或 <p> 标记。

小组件

在文档中插入小组件(Smart Widgets)。

KompoZer 制作

在当前文档，插入一个"由 KompoZer 制作" 按钮。

日历

在当前文档，插入一个日历控件。

模板

当使用模板时，您可以使用这个菜单项来插入一个可编辑区域。

PHP 代码

在 HTML 文档中，插入 PHP 代码。

注释

在 HTML 文档中，插入注释内容。

图片下换行

在图片下面插入一个换行，使得输入的文本出现在图片下方。

格式

这个菜单包含设置文本格式的各个选项。

字体

您可以为选中的文本选择字体，从系统安装的字体列表中选择所需的字体。

大小

更改字体的大小，选择较小或较大。

为选中的文本选择较小、小、中等、大、较大或者最大的文本大小。

文本样式

您可以为选中的文本选择 HTML 文本样式。

粗体

使选中的文本加粗显示，例如: 粗体文本

斜体

使选中的文本斜体显示，例如: 斜体文本

下划线

为选中的文本添加下划线，例如: 下划线文本

删除线

为选中的文本添加下划线，例如: 删除线文本

上标

使选中的文本显示为上标，例如: 上标 文本。

下标

使选中的文本显示为下标，例如: 下标文本。

固定宽度

使选中的文本固定宽度显示，例如: 固定宽度文本。

不换行

HTML 会将多个连续空格看作使一个空格，要标记不换行空格，在选中多个空格后， 使用此选项 ，例如: 三个不换行空格    到这里。

强调

强调显示文本，例如: 强调文本

特别强调

特别强调显示文本，例如: 特别强调 的文本。

引用

将选中的文本定义为引用，例如: 引用文本

缩写

指示文本是一个缩写，例如: UN

首字母缩写词

定义一个缩写词，例如: WWW

代码

定义选中的文本是计算机代码文本，例如: code text

样例输出

定义选中的文本作为一个计算机代码输出样例，例如: sample code

变量

定义选中的文本作为变量名称，例如: variable

文本颜色...

打开拾色器窗口，为选中的文本选择一个颜色。

书写方向

设定选中的文本书写方向为从左到右，或者是从右到左。选中的文本应该是块一级别的，而不是内联的文本。

取消文本样式

取消所有当前位置的文本样式。

取消链接

取消当前位置文本的链接属性。

删除锚点

删除选中的锚点。

段落

设定选中文本的样式为正文、段落、标题1到标题6,地址或预格式化文本。

列表

用一个选项来指定不同类型的列表，也就是符号列表、编号列表、组或定义， 选择列表属性打开一个对话框，手动编辑列表属性。

增大缩进

对选中的文本块增加缩进。

减小缩进

对选中的文本块减少缩进。

对齐方式

对选中的文本指定对齐方式，您可以让文本居中、左对齐、右对齐或两端对齐。

定位网格

打开一个对话框，设定移动一个层的像素值。

高级属性

打开“高级属性编辑器”，您可以手动编辑/添加 HTML 属性、内联样式或者 Javascript 事件。

背景和颜色...

打开设定页面颜色和背景的窗口，用户可以设定普通文本、链接文本，激活链接、已经访问链接 的颜色，您也可以设定页面背景色或是选择一幅背景图片。

页面标题和属性...

打开“页面属性”窗口，您可以为页面添加标题、作者和描述信息， 您也可以为当前页指定语言和书写方向。要设置此页为一个模板，可以选中“模板页”复选框。

[ 返回到开头 ]

表格

这个菜单项允许您在 HTML 文档中创建/编辑表格。

插入

打开一个子菜单，可以选择各项表格编辑/创建选项。

表格

打开一个对话框可以在当前位置选择各种表格属性。用户可以通过点击表格线设定表格大小，或者精确设定表格大小。其他单元格属性像内边距、外边距等等，也可以在这里设置。

上面一行

在当前行的 上面 插入一个空行。

下面一行

在当前行的 下面 插入一个空行。

左一列

在当前列的 左边 插入空的一列。

右边一列

在当前列的 右边 插入空的一列。

左边一格

在当前单元格的 左边 插入一个空单元格。

右边一格

在当前单元格的 左边 插入一个空单元格。

选择

选择当前表格的各个元素。

表格

选择整个表格。

行

选中当前行。

列

选中当前列。

所有单元格

选中所有单元格，这跟选中整个表格不同，此时的操作只针对所有单元格，不影响表格。

删除

打开子菜单，删除各个表格组件。

表格

删除当前的表格，会出来一个警告，所有表格数据将丢失。

行

删除当前行或选中的多行。

列

删除当前列或选中的多列。

单元格

删除当前单元格或选中的多个单元格。

单元格内容

删除当前单元格的所有内容。

向右合并单元格

合并当前单元格和它右边的单元格。

拆分单元格

拆分 colspan 值大于 1 的单元格。

从选中文本创建表格

把选中的文本转换成表格，每个段落作为一行，列的分隔可以选择用逗号(,)、空格( )，或自定义一个字符。

表格或单元格背景色...

打开一个对话框，编辑表格或单元格的背景色。

表格属性...

打开表格或单元格属性编辑窗口，您可以使用一个图形界面，更改所有表格或单元格属性。

工具

清理标记

打开清理标记窗口，可以删除多于的 <br> 标记，空的区域块等。

验证 HTML

打开一个窗口 ，通过 W3C HTML 校验网站，显示当前文档的验证结果。

CSS 编辑器

打开内置的 CSS 编辑器，编辑 CSS 样式style rules.

附加组件

打开扩展管理器，您可以安装或删除扩展，扩展可以向 KompoZer 添加新特性。

主题

打开主题管理器，您可以安装或删除 KompoZer 主题。

网站开发工具

打开各种用于网站开发的 KompoZer 工具包。

Javascript 控制台

打开 Javascript 控制台窗口。

Java 控制台

打开 Java 控制台窗口。

首选项

打开 首选项 窗口，更改个人使用 KompoZer 的选项设置。

帮助

帮助主题

打开帮助窗口。

KompoZer 网站

用默认的浏览器打开 KompoZer 网站。

常见问题(FAQ)

用默认的浏览器打开 KompoZer 的 FAQ 页面。

错误报告和意见反馈

用默认的浏览器打开 KompoZer 的错误报告和意见反馈页面。

开发者社区

用默认的浏览器打开 KompoZer 的开发者网站。

翻译 KompoZer

打开一个页面，指导您如何开始 KompoZer 的本地化进程。

论坛

用默认的浏览器打开 KompoZer 论坛站点。

日积月累

打开一个 每日提示小窗口。

关于 KompoZer

显示一个对话框，包括 KompoZer 当前版本和贡献者列表
