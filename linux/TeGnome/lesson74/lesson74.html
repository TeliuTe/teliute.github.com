七十四、终端服务客户端　返回目录　下一课

　使用它，您可以连接到远程的 Windows 和 Ubuntu 计算机；

　1、终端服务客户端

　1）点菜单“应用程序 - 互联网 - 终端服务客户端”，打开窗口；

　

　

　2）窗口中有计算机名、协议登录选项和显示等其他选项；

　

　

　3）RDP/RDPv5 用于连接 Windows 系统，使用 VNC 要安装 xvnc4viewer，使用 xdmcp 要安装 xnest 包；

　

　

　4）对方的计算机要开启远程，在 “系统 - 首选项 - 远程桌面” 中开启；

　

　

　5）输入远程计算机的登录信息，选择 VNC 协议连接 Ubutnu，点“连接”；

　

　

　6）对方允许后，就可以看到对方的桌面了，提前把窗口缩小一些；

　

　

　7）vnc密码灰色不让改，可以先选RDP协议，改完后再选 VNC 协议；

　

　本节学习了终端服务客户端的基础知识，如果你成功地完成了练习，请继续学习下一课内容；
