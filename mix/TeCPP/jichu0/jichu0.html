基础0－安装Geany　返回目录　下一课

　Geany 是一个小巧的c/c++编程平台，它也是一个GNU许可协议的软件，下面我们来看一下怎样安装；

　1、下载 Geany

　1）官方下载：http://geany.uvena.de/

　2）华军下载：http://www.newhua.com/soft/58328.htm

　2、安装 Geany

　1）在 Windows 系统中，运行下载的安装包，按提示一步一步即可，

　2）然后到 http://prdownloads.sourceforge.net/mingw 上下载 Automated MinGW Installer 自动安装包，然后一步一步运行它，

　到勾选的时候，打勾第一个 MinGW base tools 和 g++ compiler；

　

　

　3）然后点“Next”耐心等待下载、安装完成以后，到 C:\MinGW 文件夹里，复制里面的内容到 C:\Program File\geany 文件夹中；

　

　

　4）以后在保存新建的 c++ 源程序，保存到默认的文件夹，即放在 geany\bin 里；

　5）在 Ubuntu 系统中，进入新立得软件包管理器，搜索 geany 、 g++ 和 build-essential 标记安装即可；

　

　

　3、运行 Geany

　1）安装好以后，到“开始”菜单或“应用程序”菜单中，找运行的菜单；

　

　2）启动以后出来一个编辑器窗口，工作区分成了两栏，中间是输入代码的地方；

　

　　

　本节学习了安装和运行 Geany 的基本操作，如果你成功地完成了练习，请继续学习下一课内容；
