第13课 学会猜数字游戏
1、画一个标签A，文本框abc，按钮ok
2、设好属性（text），按钮MsD，第2行输：
public i as integer = int(rnd(1,10))
3、在按钮的MsD里输：
If Val(TextBox1.Text) > i Then Label1.Text = "large"
If Val(TextBox1.Text) < i Then Label1.Text = "small"
If Val(TextBox1.Text) = i Then Label1.Text = "right"
